ITC01 Java2D/3D Graphics Assignment

How to use
After compile this program by using Java 6, execute the command 'java Planet &'.
Then, you can view the planet which rotates around the axis.
If you drag mouse by left click, you can rotate the planet.
If you drag mouse by center click, you can scale up the planet.
If you drag mouse by right click, you can move the planet.
